# Hummingbird UI Template (Vite-js + Tailwind CSS)

A lightweight starter template for building modern web projects with Vite, Tailwind CSS, and Hummingbird UI.

## Quick Overview

```bash
  npx create-hummingbird-app my-app
  cd my-app
  npm run dev
```
